# Cafeteria Aconchego - Website Responsivo

> **Disciplina**: Programação Web  
> **Unidade 2 | Seção 3.1**  
> **Atividade proposta: "Cafeteria Aconchego - Website Responsivo"**

---

## Objetivo
Desenvolver um site **totalmente responsivo** para a Cafeteria Aconchego, utilizando:
- **HTML semântico** com todas as seções obrigatórias 
- **CSS Flexbox** no header e no grid de produtos
- **Media Queries** com 3 breakpoints funcionais 
- **6 produtos no cardápio** com imagem, nome, descrição e preço
- **Menu responsivo** (vertical no mobile / horizontal no desktop)
- **Cores da marca** aplicadas corretamente 
- **Meta viewport** incluída no HTML
- **Navegação interna** funcionando pelos IDs
